DTO/VO/DAO
-------------
#DTO : Data Transfer Object
>데이터를 전송할때 사용되는 오브젝트
#VO : Value Object
>DTO와 동일하지만 Read-only
#DAO : Database Access Object
>DB 접근할 때 사용하는 오브젝트
